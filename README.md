# timestacks

Wave Timestacks

# to run

1. Add a video to the repo
2. Add virtual environment if not already present: `python3 -m venv venv`
3. Activate virtual enviroment: `source venv/bin/activate`
4. Install requirements if needed: `pip install -r requirements.txt`
5. Run script: `python timestack.py VIDEO_NAME`
6. Deactivate virtual environment: `deactivate`
